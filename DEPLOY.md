# Deploy (Ayikho A/B Test v3 — Ayikho Theme)

This zip includes the app in **two places**:
- `/index.html` (root)
- `/docs/index.html` (docs)

So it works whether GitHub Pages is set to **/(root)** or **/docs**.

## Steps
1) Upload everything from this zip into your repo.
2) Settings → Pages:
   - Branch: main
   - Folder: **/(root)** (recommended) or **/docs**
3) Hard refresh:
   - Mac Chrome: Cmd + Shift + R
